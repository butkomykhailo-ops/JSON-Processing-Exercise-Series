Всі завдання спрямовані на роботу з JSON, валідацію вхідних даних та обробку HTTP статусів.

Exercise 6: JSON Echo (json_echo.js) — Базовий ендпоінт для "віддзеркалення" отриманого JSON-об'єкта.

Exercise 7: JSON Object (json_object.js) — Робота з властивостями об'єкта, генерація привітання та логічна перевірка віку.

Exercise 8: JSON Array (json_array.js) — Агрегація даних з масиву (підрахунок кількості, суми та середнього значення).

Exercise 9: JSON Calc (json_calc.js) — Реалізація динамічного калькулятора з обробкою математичних помилок (ділення на нуль).

Exercise 10: JSON Nested (json_nested.js) — Робота зі складними вкладеними структурами даних та масивами всередині об'єктів.



Ручний запуск
Кожне рішення можна запустити локально, вказавши порт як аргумент:
node json_calc.js 3000

Перевірка за допомогою curl:
curl -X POST http://localhost:3000/json-calc \
     -H "Content-Type: application/json" \
     -d '{"a": 10, "b": 5, "operation": "multiply"}'
